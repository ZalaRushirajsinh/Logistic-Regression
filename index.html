LOGISTIC REGRESSION

What is Logistic Regression?

It is a supervised machine learning algorithm used for classification problems.

Logistic Regression predicts the categorical outcome.

Like it predicts in a binary outcome, such as yes/no, pass/fail, etc...

Why is it Used?

It is used when we need to solve the classification problem where the outcome is binary.

Example: whether the person will visit the particular website or not? Is an email spam or not?

How is it Helpful in Classification?

Logistic Regression is a classification tool because it determines the probability of an event.

The primary way logistic regression helps in classification is by converting a linear equation's output into a probability score using the sigmoid function. The sigmoid function squashes any real number into a value between 0 and 1. This value represents the probability of an observation belonging to the positive class.
